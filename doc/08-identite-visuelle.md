# McClean - Identite Visuelle et Charte Graphique

## Logo

Le logo MC CLEAN est compose des lettres "MC" en bleu fonce avec "CLEAN" integre entre les deux.
Des bulles bleues claires et des etoiles scintillantes entourent le logo, evoquant le nettoyage et la proprete.

Slogan : **"Redonnez vie a vos interieurs"**

Le fichier source du logo est disponible dans `doc/logo.pdf`.

## Palette de couleurs

| Nom | Usage | Hex (approximatif) |
|-----|-------|---------------------|
| **Bleu fonce** | Couleur principale, texte du logo, boutons primaires | `#1B2A5C` |
| **Bleu moyen** | Accents, icones, elements interactifs | `#4A7EC2` |
| **Bleu clair / bulles** | Fond de decorations, bulles, illustrations | `#A8D5F2` |
| **Blanc** | Fond principal, texte sur fond sombre | `#FFFFFF` |
| **Gris clair** | Fonds secondaires, separateurs, champs de saisie | `#F2F4F7` |
| **Gris fonce** | Texte secondaire | `#6B7280` |
| **Vert** | Confirmations, succes, statut termine | `#22C55E` |
| **Rouge** | Erreurs, annulations | `#EF4444` |

> Les codes hex sont approximatifs et devront etre affines a partir du fichier logo original
> en haute resolution (demander le fichier source AI/SVG au graphiste si possible).

## Typographie (recommandation)

| Usage | Police suggeree |
|-------|-----------------|
| Titres | Inter Bold ou Poppins Bold |
| Corps de texte | Inter Regular |
| Boutons | Inter Semi-Bold |

Le choix final de la police doit etre valide avec le gerant pour rester coherent avec les supports existants (carte de visite, flyers).

## Elements graphiques

- **Bulles** : utilisees comme elements decoratifs dans les fonds et transitions
- **Etoiles scintillantes** : petites etoiles a 4 branches pour suggerer la proprete
- **Coins arrondis** : coherent avec le style moderne et accueillant du logo
- **Icones** : style outline (Lucide Icons), couleur bleu fonce ou bleu moyen
